# 📘 HR Portal Chatbot - 누구나 쉽게 따라하는 개발 가이드 (Vibe Coding)

이 문서는 개발 지식이 없는 분들도 **AI 코딩 어시스턴트(Antigravity 등)**를 활용하여 이 프로젝트를 수정하고 발전시킬 수 있도록 작성된 상세 지침서입니다.

---

## 1. 프로젝트 소개
이 프로젝트는 **사내 지식 챗봇 및 HR 포털 시스템**입니다. 직원들이 급여, 복지, 사규 등을 챗봇에게 물어보거나, 클릭 몇 번으로 확인할 수 있는 웹 사이트입니다.

### 🛠 사용된 기술 (알아두면 좋은 용어)
AI에게 작업을 지시할 때 이 용어들을 사용하면 더 정확하게 알아듣습니다.
- **React (리액트)**: 화면을 만드는 레고 블록 같은 도구입니다.
- **TypeScript (타입스크립트)**: 코드의 오류를 미리 잡아주는 안전장치입니다.
- **Vite (비트)**: 작업한 결과물을 화면에 아주 빠르게 보여주는 도구입니다.
- **Tailwind CSS**: 디자인을 꾸미는 도구입니다.
- **Google Gemini API**: 챗봇의 두뇌 역할을 하는 AI입니다.

---

## 2. 준비물 (개발 환경 설정)
이 프로젝트를 실행하기 위해 딱 두 가지만 설치해주세요.

1.  **Node.js (노드 JS)**
    *   [Node.js 공식 홈페이지](https://nodejs.org/)에서 'LTS' 버전을 다운로드하여 설치하세요.
    *   설치 후 터미널(명령 프롬프트)에서 `node -v`를 입력했을 때 숫자가 나오면 성공입니다.

2.  **VS Code (비주얼 스튜디오 코드)**
    *   [VS Code 홈페이지](https://code.visualstudio.com/)에서 다운로드하여 설치하세요.
    *   이곳에서 AI 어시스턴트와 대화하며 코드를 수정하게 됩니다.

---

## 3. 프로젝트 시작하기 (3단계)

**1단계: 프로젝트 폴더 열기**
VS Code를 실행하고 `파일(File) > 폴더 열기(Open Folder)`를 눌러 이 프로젝트 폴더를 선택하세요.

**2단계: 설치하기**
VS Code 상단 메뉴에서 `터미널(Terminal) > 새 터미널(New Terminal)`을 누르고 아래 명령어를 입력 후 엔터(Enter)를 치세요.
```bash
npm install
```
*(처음 한 번만 하면 됩니다. 필요한 부품들을 다운로드하는 과정입니다.)*

**3단계: 실행하기**
설치가 끝나면 아래 명령어를 입력하세요.
```bash
npm run dev
```
잠시 후 `Local: http://localhost:5173/` 같은 주소가 나옵니다. `Ctrl` 키를 누른 채 저 주소를 클릭하면 웹 사이트가 열립니다!

## 4. 바이브 코딩 (AI와 함께 개발하기)
이제 직접 코드를 짤 필요가 없습니다. **"무엇을 하고 싶은지"** AI에게 명확하게 말하면 됩니다.

### 💡 AI에게 작업 시키는 법 (프롬프트 예시)

**상황 1: 디자인을 바꾸고 싶을 때**
> "현재 파란색인 헤더 배경색을 우리 회사 브랜드 컬러인 짙은 주황색(#FF5722)으로 바꿔줘. 그리고 글자색은 흰색으로 해줘."

**상황 2: 새로운 메뉴를 추가하고 싶을 때**
> "사이드바 메뉴에 '구내식당' 아래에 '셔틀버스 시간표' 메뉴를 추가해줘. 아이콘은 버스 모양으로 해주고, 클릭하면 버스 노선표 이미지를 보여주는 화면을 만들어줘."

**상황 3: 기능이 작동하지 않을 때 (버그 수정)**
> "지금 챗봇한테 질문을 했는데 답변이 안 와. 브라우저 콘솔에 빨간색 에러가 뜨는데 확인해서 고쳐줄래?"

**상황 4: 새로운 기능을 넣고 싶을 때**
> "공지사항 게시판에 '작성자' 이름도 같이 표시되게 수정해줘. 그리고 최신 글이 맨 위로 오도록 정렬해줘."

### 🌟 꿀팁: 더 잘 시키는 노하우
1.  **구체적으로 말하기**: "예쁘게 해줘"보다는 "모서리를 둥글게 하고 그림자를 넣어줘"가 좋습니다.
2.  **하나씩 시키기**: 한 번에 너무 많은 걸 시키면 AI도 헷갈립니다. "메뉴 추가해줘" -> (완료 후) -> "이제 그 메뉴 디자인 바꿔줘" 순서로 진행하세요.
3.  **파일 위치 알려주기 (몰라도 됨)**: AI가 알아서 찾지만, "사이드바 파일(`Sidebar.tsx`)에서 수정해줘"라고 하면 더 빠릅니다.

## 5. 폴더 구조 (지도)
혹시 파일을 직접 찾아보고 싶다면 이 지도를 참고하세요.

- `src/` : 모든 소스 코드가 여기 있습니다.
  - `App.tsx` : 웹 사이트의 뼈대, 가장 중요한 파일입니다.
  - `components/` : 화면 구성 요소들 (레고 블록)이 모여 있습니다.
    - `Sidebar.tsx`: 왼쪽 메뉴바
    - `ChatInterface.tsx`: 챗봇 대화 화면
    - `Login.tsx`: 로그인 화면
  - `services/` : 챗봇 연결 등 복잡한 기능들이 들어있습니다.
  - `constants.ts` : 메뉴 이름, 공지사항 내용 등 고정된 글자들은 여기서 쉽게 바꿀 수 있습니다.

## 6. 문제 해결 (FAQ)

**Q. `npm run dev`를 했는데 에러가 나요.**
A. 터미널에 `npm install`을 다시 한 번 입력해보세요. 그래도 안 되면 AI에게 에러 메시지를 복사해서 "이 에러 해결해줘"라고 물어보세요.

**Q. AI가 코드를 고쳤는데 화면이 이상해요.**
A. "방금 수정한 거 취소해줘" 또는 "화면이 깨졌어, 다시 원래대로 돌려줘"라고 말하면 됩니다.
